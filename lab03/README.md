## **Задание 1**

**Задание:** Написать функцию для поиска значения в сложном объекте.

**Ход выполнения:** 
1. Рекурсивная функция: обходим список, увеличиваем счетчик только для элементов-значений, при встрече списка рекурсивно заходим в него и прибавляем результат к текущей позиции.
2. Нерекурсивная функция: используем s для хранения списков и их начальных позиций, последовательно обрабатываем все элементы.
