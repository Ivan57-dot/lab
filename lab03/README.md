## **Задание 1**

**Задание:** Написать функцию для поиска значения в сложном объекте.

**Ход выполнения:** 
1. Рекурсивная функция: обходим список, увеличиваем счетчик только для элементов-значений, при встрече списка рекурсивно заходим в него и прибавляем результат к текущей позиции.
2. Нерекурсивная функция: используем s для хранения списков и их начальных позиций, последовательно обрабатываем все элементы.

## **Задание 2**

**Задание:** Написать функцию для расчета пои реккурентной формуле.

**Ход выполнения:** 
1. Рекурсивная функция: использует базовый случай k=1, а для остальных случаев вызывает саму себя для предыдущего шага.
2. Нерекурсивная функция: инициализирует a и b начальными значениями, в цикле от 2 до k вычисляет новые значения по формулам, на каждой итерации обновляет a и b, после цикла возвращает результаты для заданного k.
