## **Задание 1**

**Задание:** Замыкание для накопления всех аргументов в коллекции, при получении определённого значения - возврат и очистка коллекции.

**Ход выполнения:** 
1. Создана внешняя функция make_zamikanie(), внутри которой определен пустой список items для накопления аргументов.
2. Определена внутренняя фнукция zamikanie(), которая имеет доступ к списку items:
     - Если аргумент равен "стоп", функция возращает копию накопленного списка и очищает его.
     - Если аргумент другой - добавляет его в список.
3. Возращение внутренней функции из внешней, чтобы создать замыкание.
