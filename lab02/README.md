# **ОТЧЕТ ПО ЛАБОРАТОРНОЙ РАБОТЕ №2**
## **Задание 1**
**Задание:** Иван составляет 5-буквенные коды из букв И, В, А, Н. Буквы в коде могут повторяться, использовать все буквы не обязательно, но букву И нужно использовать хотя бы один раз. Сколько различных кодов может составить Иван? 
**Ход выполнения:**
1. Сгенерировать все 5-буквенные комбинации из букв И, В, А, Н
2. Проверить наличие буквы 'И' в каждой комбинации
3. Подсчитать количество подходящих комбинаций
